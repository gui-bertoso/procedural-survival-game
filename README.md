# procedural survival game

a fully procedural survival game built with godot.

focused on performance, scalability and real-time world generation.

---

## features

- chunk-based world streaming  
- procedural terrain generation  
- optimized vegetation using multimesh  
- level of detail (lod) system  
- scalable architecture for large worlds  

---

## technical highlights

- dynamic chunk loading/unloading  
- object pooling to reduce allocations  
- multimesh instancing (massive draw call reduction)  
- async-friendly architecture  

---

## goal

build a scalable survival game capable of generating large worlds in real time.

---

## stack

- godot (gdscript / c#)
- procedural generation
- performance optimization
